NiklasRosenstein.cpp-nr

A simple C++ library with various components, complete with tests, benchmarks, examples and continous integration. Demonstrates how to use Google Test and Google Benchmark and how to implement a re-usable Craftr package. The library can easily be used by using its Craftr Framework.

cxx = load('craftr.lang.cxx')
cpp_nr = load('NiklasRosenstein.cpp-nr').nr

main = cxx.executable(
  output = 'main',
  inputs = cxx.compile_cpp(
    sources = glob('src/*.cpp'),
    frameworks = [cpp_nr]
  )
)
